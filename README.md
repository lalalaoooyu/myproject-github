# MyProject-GitHub

这是一个练习用的 Python 项目，用于学习：

- Git / GitHub 基本操作
- `.gitignore` 配置
- GitHub Actions 自动测试工作流
- 自动化测试（使用 pytest）
- 后续会部署到 Render/Heroku

## 📁 项目结构
```
myproject-github/
│
├── app.py # 主应用文件
├── test_app.py # 测试文件
├── requirements.txt # 依赖列表
├── .gitignore
└── .github/workflows/python-test.yml # GitHub Actions 配置
```

## 🧪 测试方法

```bash
pytest
```
